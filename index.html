Job Board – Plataforma de bolsa de empleo

Plataforma de bolsa de empleo con tres tipos de usuarios (candidato, empresa, admin), que permite publicar ofertas, postularse y moderar el contenido en tiempo real. Desarrollada con Django REST Framework, PostgreSQL, React, TypeScript y Tailwind, desplegada en Render y Netlify.

  • Frontend: React, TypeScript, Vite, Tailwind CSS
  • Backend: Django, Django REST Framework, PostgreSQL, JWT
  • Infraestructura: Render (backend + base de datos), Netlify (frontend)
  • Base de datos: PostgreSQL gestionada en Render (servicio Postgres conectado al Web Service via DATABASE_URL)

  • Rol candidato: Registro/login JWT, búsqueda y filtrado de ofertas, postulaciones con carta y sección “Mis postulaciones”.
  • Rol empresa: Panel privado, CRUD de sus ofertas, gestión de postulaciones (aceptar/rechazar).
  • Rol admin: Panel propio en el frontend, gestión de empresas (bloquear/desbloquear) y ofertas (activar/desactivar).
